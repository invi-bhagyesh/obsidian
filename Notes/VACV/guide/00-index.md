## How to Use This Guide

1. **Read each topic file sequentially** — definitions build on earlier material
2. **Work every example yourself** before reading the solution
3. **Attempt practice problems** at the end of each topic
4. **Use the consolidated problem sets** (files 07 and 18) for timed exam practice
5. **Key theorems to memorize**: Green's, Stokes', Divergence Theorem, Cauchy-Riemann equations, Cauchy's integral formula, Residue theorem
